📊 Sales Analytics Dashboard – Power BI
📌 Overview
Built an interactive Sales Analytics Dashboard in Power BI to analyze sales performance, revenue trends, and regional insights for data-driven decision-making.
🛠️ Tools
Power BI
Power Query (Folder Injection)
DAX
📈 Key Features
KPI tracking (Revenue, Profit, Sales)
Region-wise performance analysis
Monthly & yearly trend analysis
Top N products and categories
Interactive filters and drill-downs
🧠 Insights
North region drives highest revenue
Clear seasonal sales trends observed
Top products contribute major revenue share
💡 Impact
Enabled faster insights, improved reporting efficiency, and better business decisions.
